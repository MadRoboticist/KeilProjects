Detailed Description

This source file contains a c program that reads raw analog data out of the built in accelerometer in the FRDM-KL25Z board. Based on the x, y, and z position the built in RGB LED will change color. This project also utilizes the built in capacitive touch slider to control the RGB LED dimmability.

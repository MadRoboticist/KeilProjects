◆ UART_put()

UART_put ( UCHAR  )

Puts a byte into the transmit buffer.

Puts a byte, to the transmit buffer at the location pointed to by tx_in_idx. The pointer is incremented circularly as described previously. If the transmit buffer should wrap around (should be designed not to happen), data will be lost. The serial interrupt must be temporarily disabled since it reads tx_in_idx and this routine updates tx_in_idx which is a 16 bit value.
